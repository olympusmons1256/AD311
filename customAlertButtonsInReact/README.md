
## Custom Alert Buttons React App
A simple React app that features a Toolbar component with multiple AlertButton components. Each button displays a unique alert message when clicked.

### Features
- Reusable AlertButton component that accepts a custom message and children prop.
- Toolbar component renders multiple AlertButtons with unique messages and labels.
- Demonstrates passing props, using the children prop, and handling onClick events in React.

### Prerequisites
- Node.js v18 or higher recommended
- npm (comes bundled with Node.js)

### Steps to Start
- Install dependencies: `npm install`
- Start the development server: `npm run dev`
- Open your browser to the local address provided (usually http://localhost:5173)

### Usage
- Click any button in the toolbar to see a browser alert with a unique message for that button.
