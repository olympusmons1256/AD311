## Simple Text Editor With Undo Functionality


### Features
- Add text
- delete text
- undo last operation

### Steps to Start
- npm run dev ("Dev": "Vite")

### Time & Space Complexity
- Time: 0(1)
- Space: 0(1)

### Test Cases
normal
- add text
- delete text
- undo text

edge
- undo 3-times concurrently
- add text after 3 undos
- delete text after 3 undos
