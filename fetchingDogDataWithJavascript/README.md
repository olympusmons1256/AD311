## Fetching Dog Data with JavaScript
A web application that demonstrates using the JavaScript Fetch API to retrieve and display data from the Dog API.

### Features
- Fetch and display a list of dog breeds
- View detailed information about specific breeds
- Load random dog facts
- Display dog breed groups
- Interactive breed selection with click events

### Prerequisites
- A modern web browser

### Steps to Start
- Open index.html in your web browser
- Click the buttons to load data from the Dog API

### API Usage
- GET https://dogapi.dog/api/v2/breeds // Returns all dog breeds
- GET https://dogapi.dog/api/v2/breeds/{id} // Returns specific breed details
- GET https://dogapi.dog/api/v2/facts?limit=5 // Returns dog facts
- GET https://dogapi.dog/api/v2/groups // Returns dog breed groups
