## Recipe Gallery React App
A simple React application that displays a gallery of curated recipes using static data.

### Features
- View a list of recipes with images, titles, and ingredients
- Responsive card layout using CSS Flexbox
- No state management or interactivity (static rendering)

### Prerequisites
- Node.js v18 or higher recommended
- npm (comes bundled with Node.js)

### Steps to Start
- Install dependencies: `npm install`
- Start the development server: `npm start`
- Open your browser at http://localhost:3000 to view the gallery

### Project Structure
- `src/RecipeGallery.jsx`: Main component rendering the recipe cards
- `src/RecipeGallery.css`: Styling for the gallery and cards
- './assets/[image...]' : contains all of the gallery assets (meant to setup for CDN structure in future)
- `public/`: Contains favicon and logo images
