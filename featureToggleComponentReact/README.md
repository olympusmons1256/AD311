
## Feature Toggle React App
A simple React application that demonstrates conditional rendering using a reusable Feature Toggle component.

### Features
- Enable or disable features using props
- Reusable `FeatureToggle` component for any feature
- No state management (static rendering based on props)

### Prerequisites
- Node.js v18 or higher recommended
- npm (comes bundled with Node.js)

### Steps to Start
- Install dependencies: `npm install`
- Start the development server: `npm start`
- Open your browser at http://localhost:3000 to view the demo
